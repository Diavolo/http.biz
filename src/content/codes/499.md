# This is not a status code available in the standard, but it exists
# with significant prominence in the wild
set: 4
code: 499
title: Client Closed Request
---

Un código de estado no estándar introducido por [nginx][2] para el caso cuando un cliente cierra la conexión mientras nginx está procesando la solicitud.
