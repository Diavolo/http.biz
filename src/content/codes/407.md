Similar a [401 Unauthorized](/401), pero indica que el cliente necesita autenticarse para usar un proxy.

El proxy DEBE enviar un campo de encabezado Proxy-Authenticate<sup>[1](#ref-1)</sup> que contenga un desafío aplicable a ese proxy para el recurso objetivo. El cliente PUEDE repetir la solicitud con un campo de encabezado Proxy-Authorization nuevo o reemplazado<sup>[2](#ref-2)</sup>.
